Uvod u CSS

 

CSS je kod koji definiše izgled HTML elemenata tj. njihovu boju, dužinu, širinu, margine, ivice… Ovaj kod se ne može samostalno koristiti, jer on samo dopunjava HTML kod. Sve stranice na internetu koriste CSS uz HTML.


CSS(Cascading Style Sheets) je posebni deo HTML-a koji definiše izgled stranice. Sa CSS-om može menjati izgled stranice odnosno menjati elemente stranice kao što su boja, font, veličina, margine itd.
CSS je vrlo jednostavan za učenje jer se sastoji od jenog šablona koji podrazumeva selektore i parametre. Selektorima govorimo browser-u koji element želimo da izmenimo pomoću css-a, a parametrima definišem njegov izgle odnosno podešavamo boju, veličinu, pozadinu…


CSS kod možemo koristiti unutar HTML dokument pod tagovima style ili kao posebni dokument sa ekstenzijom .css. U html uključujemo css dokumente pomoću head taga.


Istroija CSS kaže da je CSS u određenoj formi bio zastupljen čak 70-ih godina. Kako je HTML bivao sve komplikovaniji 1996. godine u decembru objavljen je prvi CSS1.
